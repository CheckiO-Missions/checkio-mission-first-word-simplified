Прежде всего, попробуйте использовать метод str.replace() чтобы заменить точки и запятые пробелами.
Вы могли бы показать, как это реализовать в коде?
В данном случае код может выглядеть так: 
text = text.replace('.', ' ')
text = text.replace(',', ' ')
Например, если наш исходный текст:
text = ".,.hello dude.,."
мы используем код, написанный выше и получаем:
" hello dude "
Мы специально заменяем точки и запятые на пробелы, а не убираем их (заменяя на пустые строки - ""), чтобы избежать ситуации объединения двух слов в одно. Например, если бы наш исходный текст выглядел как "hi,dude", при замене знаков препинания на пробелы мы получим "hi dude", где легко можно будет определить первое слово. Если же заменить запятую на "", то результат будет - "hidude" - одно цельное слово. Также, этот код может быть записан в одну строку: 
text = text.replace('.', ' ').replace(',', ' ')

Последнее, что вам необходимо сделать - найти первый элемент списка. Это можно сделать, обратившись к нему по индексу, например так - words[0]. Обратите внимание, что нумерация начинается не с 1, а именно с 0 - это очень важный момент в Python.
Как будет выглядеть программа полностью?
Полная версия кода выглядит так: 
def first_word(text):
    text = text.replace('.', ' ').replace(',', ' ')
    text = text.split()
    return text[0]
