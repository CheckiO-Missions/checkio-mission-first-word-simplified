Vous donnez une chaîne et vous devez en trouver le premier mot.

Il est une version plus simple de la mission Premier Mot, qui peut être complétée plus tard.

  • La chaîne d'entrée se compose uniquement de lettres et d'espaces anglais.
  • Il n'y a pas d'espaces au début et à la fin de la chaîne.

En entrée: Une chaîne.

En sortie: Une chaîne.

Example:

{% if interpreter.slug == "js-node" %} 
firstWord("Hello world") == "Hello"
{% else %} 
first_word("Hello world") == "Hello"
{% endif %}

Comment il est utilisé: Le premier mot est une commande dans une ligne de commande.

Condition préalable: Le texte peut contenir a-z, A-Z et des espaces.
