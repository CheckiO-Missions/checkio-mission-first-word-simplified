D'ACCORD. Peut-être pouvez-vous me donner des liens vers des documents?

Bien sûr, si vous devez diviser une chaîne de caractères, vous devez faire string.split().

OK. Comment puis-je obtenir les mots maintenant?

En fait, en appelant cette fonction avec un espace vide comme argument. Regardez ça:

const text = "greeting from CheckiO Planet";
const words = text.split(" ");
console.log(words);

Sortie:

'greeting', 'from', 'CheckiO', 'Planet'

Bien... Comment puis-je obtenir le premier mot alors?

Le résultat de la fonction split() est un tableau de mots. Pour obtenir le premier mot, vous devez utiliser l'index 0.

console.log(words[0]);

Résultat:

'greeting'

Je n'ai toujours aucune idée de la manière dont tout cela s'articule.

OK, votre solution finale devrait ressembler à ceci:

function firstWord(text: string): string {
    
    const words = text.split(" ");
    
    return words[0];
}
