Otrzymują Państwo ciąg znaków i muszą znaleźć jego pierwsze słowo.

  • Ciąg wejściowy składa się tylko z angielskich liter i spacji.
  • Na początku i na końcu ciągu nie ma żadnych spacji.

Dane wejściowe: A {% if is_js %}string{% else %} string (str){% endif %}.

Dane wyjściowe: A {% if is_js %}ciąg (string){% else %} ciąg (str){% endif %}.

Przykłady: 

{{init_code_tmpl}}

Sposób użycia: Pierwsze słowo jest poleceniem w wierszu poleceń.

Warunek wstępny: Tekst może zawierać litery a-z, A-Z i spacje.
