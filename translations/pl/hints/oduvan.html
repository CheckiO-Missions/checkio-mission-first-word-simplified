Twoja strategia powinna opierać się na podzieleniu stringa na słowa, a następnie wybranie pierwszego z nich.
OK. Masz może jakiś link do dokumentacji?
Pewnie. Jeśli chcesz podzielić string powinieneś użyć str.split. W dokumentacji znajdziesz więcej informacji.
OK. Skąd w takim razie wziąć słowa?
Cóż, właściwie wystarczy wywołać tę funkcję. Spójrz: 
text = "greeting from CheckiO Planet"
words = text.split()
print(words)
Wyjście dla tego fragmentu kodu będzie wyglądało następująco: 
['greeting', 'from', 'CheckiO', 'Planet']
Dobrze... To jak teraz mogę wybrać pierwsze słowo?
Wynikiem funkcji split jest lista słów. Aby dostać się do pierwszego z nich, będziesz musiał użyć indeksu 0. 
print(words[0])
Nadal nie wiem jak to wszystko połączyć.
Okej. Ostateczne rozwiązanie powinno wyglądać tak: 
def first_word(text: str) -> str:
    """
        returns the first word in a given text.
    """
    return text.split()[0]
