Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Pana ogólna strategia powinna polegać na podzieleniu ciągu na słowa, a następnie zdobyciu pierwszego z nich.

OK. Może poda mi Pan jakieś linki do dokumentów?

Oczywiście, jeśli musi Pan podzielić ciąg znaków, powinien Pan to zrobić string.split().

OK. Jak mogę teraz uzyskać słowa?

Cóż, w rzeczywistości, wywołując tę funkcję z pustym argumentem spacji. Proszę to sprawdzić:

const text = "greeting from CheckiO Planet";
const words = text.split(" ");
console.log(words);

Output:

'greeting', 'from', 'CheckiO', 'Planet'

Cóż... Jak zatem mogę uzyskać pierwsze słowo?

Wynikiem funkcji split() jest tablica słów. Aby uzyskać pierwsze z nich, należy użyć indeksu 0.

console.log(words[0]);

Wynik:

'greeting'

Nadal nie mam pojęcia, jak to wszystko połączyć.

W takim razie ostateczne rozwiązanie powinno wyglądać następująco:

function firstWord(text: string): string {
    
    const words = text.split(" ");
    
    return words[0];
}
