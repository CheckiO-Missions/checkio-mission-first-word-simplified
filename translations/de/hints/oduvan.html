Ihre allgemeine Strategie sollte darin bestehen, die Zeichenkette in Wörter aufzuteilen und dann das erste Wort zu finden.

GUT. Vielleicht können Sie mir ein paar Links zu Dokumenten geben?

Sicher, wenn Sie eine Zeichenkette aufteilen müssen, sollten Sie str.split() verwenden.

NUN GUT. Wie kann ich jetzt die Wörter erhalten?

Nun, eigentlich durch den Aufruf dieser Funktion. Sehen Sie sich das an:

text = "greeting from CheckiO Planet"
words = text.split()
print(words)

Ausgabe:

['greeting', 'from', 'CheckiO', 'Planet']

Nun... Wie kann ich dann das erste Wort erhalten?

Das Ergebnis der Funktion split() ist also eine Liste von Wörtern. Um das erste Wort zu erhalten, müssen Sie den Index 0 verwenden.

print(words[0])

Ausgabe:

'greeting'

Ich habe immer noch keine Ahnung, wie ich das alles zusammensetzen soll.

OK, dann sollte Ihre endgültige Lösung wie folgt aussehen:

def first_word(text: str) -> str:
    
    words = text.split()

    return words[0]
