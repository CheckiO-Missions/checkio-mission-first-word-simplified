Твоя загальна стратегія тут - це розбити рядок на слова і тоді взяти перше із них.

Зрозуміло. Може дасиш якесь посилання на відповідні матеріали?

Звісно! Якщо тобі потрібно розбити рядок, то слід використати string.split().

Добре. То як мені власне отримати ці слова?

Взагалі то, викликавши вищенаведену функцію. Ось, поглянь:

const text = "greeting from CheckiO Planet";
const words = text.split(" ");
console.log(words);

Вивід:

'greeting', 'from', 'CheckiO', 'Planet'
І як мені взяти звідси перше слово?

Отже, результатом функції split() є масив зі слів. Щоб взяти перше із них, тобі потрібно звернутися до значення із індексом 0.

console.log(words[0]);

Вивід:

    'greeting'
Я все ще не розумію, як застосувати це разом.

Твій фінальний розв'язок матиме вигляд:

import assert from "assert";


function firstWord(text: string): string {
    
    const words = text.split(" ");
    
    return words[0];
}
