当然，如果你需要分割一个字符串，你应该做string.split() 。

好的。我现在怎么能得到这些词呢？

嗯，实际上，通过调用这个带有空位参数的函数。看看这个吧：

const text = "greeting from CheckiO Planet";
const words = text.split(" ");
console.log(words);

输出：

'greeting', 'from', 'CheckiO', 'Planet'

那么......我怎样才能得到第一个词呢？

所以，split() 函数的结果是一个单词数组。为了得到第一个词，你需要使用索引0 。

console.log(words[0]);

输出：

'greeting'

我还是不知道如何把这些东西放在一起。

好吧，你的最终解决方案应该是这样的：

function firstWord(text: string): string {
    
    const words = text.split(" ");
    
    return words[0];
}
