文字列が与えられます。その中から先頭の単語を見つけ出しましょう。

問題を解く際に、次の点に注意しましょう。

  • 文字列がピリオドやコンマを含むかもしれません。
  • 文字列が文字で始まるかもしれないし、ピリオドやスペースで始まるかもしれません。
  • 単語はアポストロフィーを含むことが出来ます。そして、それは単語の一部です。
  • テキスト全体を一単語で表すことが出来ます。

入力: 文字列

出力: 文字列

例:

{% if interpreter.slug == "js-node" %} 
firstWord("Hello world") == "Hello"
firstWord("greetings, friends") == "greetings"
{% else %} 
first_word("Hello world") == "Hello"
first_word("greetings, friends") == "greetings"
{% endif %}

どのように使えるか: コマンドラインの先頭の単語はコマンドです。

事前条件: テキストは a-z A-Z , . ' を含むことが出来ます。
